# Домашнее задание 5: Освоение Claude Code и Мультиагентных систем

На основе пятой лекции вам предстоит перенести часть своей рутины на новый инструмент и попробовать поработать с мультиагентной системой.

## Основное задание

### Шаг 1. Развертывание Claude Code
1. Откройте новый терминал в вашей среде разработки.
2. С помощью команд из лекции разверните **Claude Code** (Claude CLI).
3. Пройдите процесс авторизации, привязав свой аккаунт с активной подпиской Anthropic (или используя API-баланс).

### Шаг 2. Запуск роя агентов для аудита проекта
1. Откройте папку с вашим текущим рабочим проектом (открытым в терминале).
2. Напишите Claude Code промпт для запуска нескольких агентов параллельно. Например:
   > «Запусти 4 субагента для моего проекта. Первый пусть ищет потенциальные баги. Второй проверит покрытие кода. Третий напишет краткую документацию по функционалу проекта. Четвертый составит список необходимых доработок как Senior Разработчик».
3. Дождитесь, пока все агенты завершат работу, и внимательно изучите итоговый лог/сводку.
4. Выявленные ошибки или рекомендации по улучшению кода перенесите в Antigravity для исправления.

### Шаг 3. Использование Skill Creator
1. Перезапустите терминал, если это необходимо, и установите плагин **Skill Creator** *глобально* (Global scope).
2. Вызовите Skill Creator (через `/skill`) и попросите его создать скилл для **качественного ревью кода (Code Review)** или **помощи в создании UI-интерфейсов**.
3. Дождитесь, пока Skill Creator проведет A/B тестирование вариантов вашего навыка и выберет лучший.
4. Проверьте работу созданного скилла на любом файле из вашего проекта.

## Дополнительное задание со звездочкой (*)

### Опционально: Альтернативные среды и голос
1. **Интеграция Gemini:** Установите расширение Google Gemini в вашей среде разработки (по аналогии с тем, как это обсуждалось на уроке) и авторизуйтесь. Сравните его интерфейс с Claude.
2. **Голосовое управление:** Установите консольную или системную утилиту для голосового набора текста (например, **Glider**) и попробуйте надиктовать промпт для Antigravity или Claude Code голосом. 

### Как сдавать домашнее задание:
Результатом вашей работы будет:
* Скриншот успешной работы параллельных агентов в терминале с Claude Code.
* Скриншот итогового отчета созданного скилла по тестированию вашего кода.
* (Если делали доп. задание) Короткое видео или скриншот работы голосового ввода / интерфейса Gemini.
