## Ключевые расчёты

### 1. Средний чек (ARPU)

```
ARPU = Выручка / Кол-во подписок
ARPU = 2 000 000 ₽ / 709 = 2 821 ₽
```

> ⚠️ **Внимание:** Если в выручку входят тест-драйвы — нужно их вычесть для чистоты расчёта.

---

### 2. LTV (пожизненная ценность клиента)

**Формула:** `LTV = ARPU × Среднее кол-во покупок`

При Retention 35.8%:
```
Среднее кол-во покупок = 1 / (1 - 0.358) = 1 / 0.642 = 1.56 покупок
```

```
LTV = 2 821 ₽ × 1.56 = 4 400 ₽
```

> 💡 Это значит: один платящий клиент в среднем принесёт **4 400 ₽** за всё время.

---

### 3. Максимальный CAC (стоимость привлечения)

**Правило:** CAC должен быть < LTV (желательно в 3 раза меньше для здоровой экономики)

```
Безопасный CAC = LTV / 3 = 4 400 ₽ / 3 = 1 466 ₽
```

> ✅ Можно тратить до **1 466 ₽** на привлечение одного платящего клиента и оставаться в плюсе.
